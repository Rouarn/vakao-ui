安装 ​

环境支持 ​

Vakao UI 需要 Node.js 版本 >= 16。

版本 ​

Vakao UI 目前还处于开发中，版本为 0.0.1。

使用包管理器 ​

我们建议您使用包管理器（如 NPM、Yarn 或 pnpm）安装 Vakao UI，然后您就可以使用打包工具，例如 Vite 或 webpack。

bash
# 选择一个你喜欢的包管理器

# NPM
$ npm install vakao-ui --save

# Yarn
$ yarn add vakao-ui

# pnpm
$ pnpm add vakao-ui
